## Project Overview
A database-driven information system with frontend integration demonstrating:  
- Proper normalization (1NF–3NF)  
- Referential integrity & constraints  
- Transactions (ACID) and triggers  
- Indexing & performance optimization  
- Role-based access  
- Handling 100,000+ rows dataset

- **Database:** MySQL (XAMPP)  
- **Backend:** PHP  
- **Frontend:** HTML / CSS / JS
